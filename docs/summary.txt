클로드 코드에서 에이전트와 스킬을 만들 수 있게 프롬프트를 만들어줘. 
내가 만들려고 하는 프로그램은 주식분석 프로그램이야. 
이 프로그램은 다음과 같은 원리로 작동해. 

[ 테마 -> 종목 분석 ]

1. 테마분석
 - 증권회사의 api 중 테마와 테마에 속한 주식정보를 불러온다. 
 - 테마 중 옵션을 선택하여 옵션에 해당하는 방식으로 테마를 추천 받는다. 
    1) 거래량 : 가장 거래량이 활발한 테마의 순위를 선정하고, 상위 5개 테마를 추천받는다.
    2) 급등주 : 최근 급등주가 많은 테마 5개를 추천받는다.
 - 원하는 테마를 추가할 수도 있다.
 - 추천받은 테마들을 바구니에 담는다.

2. 주식분석
 - 바구니에 담겨진 테마를 불러온다. 
 - 담겨진 테마와 관련한 뉴스 api와 웹 검색을 통해 해당 테마에 대한 최신 트렌드를 분석하여 보고서를 만든다.(테마별로 각각 선택하여 제작)
 - 각 테마 별 대표 대장주(가장 거래량이 활발한 종목부터 상위 5개) + 관련 ETF(가장 거래량이 활발한 종목부터 상위 3개)를 추천받는다.
 - 각 종목별 상세 정보를 확인한다.(데이터 표로 나타나고, 각 필드별로 엑셀처럼 다중 정렬이 가능하다.)
 - 상세정보는 다음과 같다. 
    1) 그래프(기준가, 이동평균선, 거래량)
    2) 기준가(현재가)
    3) 거래량
    4) 시가총액
    5) 외국인 거래량
    6) 기관 거래량
    7) 개인 거래량
    8) PER, PBR
    9) 동일업종 PER
    10) 배당수익률
    11) 종목뉴스
 - 원하는 종목을 선택하여 분석 보고서를 제작.
