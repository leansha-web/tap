# PRD (제품 요구사항 정의서)

**프로젝트**: Theme Analysis Program (TAP)  
**버전**: v1.0  
**작성일**: 2026-02-10  
**세션 ID**: TAP-2026-02-10

## 1. 프로젝트 개요

### 프로젝트 이름
**Theme Analysis Program (TAP)**

### 한 줄 요약
주식 투자자를 위해 테마별 종목 흐름을 한눈에 보여주는 도구

### 목표
테마 분석에 쓰는 시간을 절반으로 줄이고, 더 확신 있게 투자 결정을 내리게 한다.

---

## 2. 문제 정의

### 현재 상황
- 요즘 테마주가 빠르게 변하는데, 그 큰 흐름을 탈 수 있는 종목들을 분별하기 어렵다.
- 한 테마 안에서도 종목이 수십 개인데, 어떤 종목이 대장주인지 찾기 어렵다.
- 기존 증권사 앱은 테마 메뉴가 있지만 종목만 나열되고, 정렬이나 비교가 불가능하다.

### 해결할 문제
1. **테마 내 대장주 발굴 어려움** — 수십 개 종목 중 진짜 리더를 찾아내는 것
2. **테마 × 종목 이중 필터링 필요** — 큰 흐름의 테마를 고르고, 그 안에서도 실패 확률이 낮은 종목을 선별하는 것

---

## 3. 사용자 페르소나

### 주요 사용자
- **연령**: 30~40대
- **직업**: 직장인
- **투자 경험**: 2~3년
- **투자 스타일**: 테마주 위주
- **사용 패턴**: 퇴근 후 스마트폰으로 주식 체크

### 사용 장면
1. **⚡ 아침 빠른 탐색** — 출근 전 뉴스를 보다가 '이 테마가 뜨겠다' 싶으면 관련 종목을 빠르게 찾고 싶을 때 (속도가 핵심)
2. **📊 주말 복기 분석** — 한 주간 테마 흐름을 되돌아보며 다음 주 전략을 세우는 모드 (깊이가 핵심)

---

## 4. 사용자 스토리

### FEAT-1: 테마 분석 & 추천
- **As a** 테마주 투자자
- **I want to** 거래량·급등주 기준으로 상위 테마 5개를 자동 추천받고, 원하는 테마를 수동으로 추가하여 바구니에 담고 싶다
- **So that** "어떤 테마가 지금 뜨는지" 빠르게 파악할 수 있다

### FEAT-2: 종목 심층 분석 & 비교
- **As a** 테마주 투자자
- **I want to** 테마별 대장주 5개 + ETF 3개를 추천받고, 11개 상세 지표를 엑셀처럼 다중 정렬 가능한 표로 보고 싶다
- **So that** "어떤 종목을 살지" 데이터 기반으로 판단할 수 있다

**11개 상세 지표**:
1. 그래프 (기준가, 이동평균선, 거래량)
2. 기준가 (현재가)
3. 거래량
4. 시가총액
5. 외국인 거래량
6. 기관 거래량
7. 개인 거래량
8. PER, PBR
9. 동일업종 PER
10. 배당수익률
11. 종목뉴스

### FEAT-3: AI 트렌드 보고서 (후순위)
- **As a** 테마주 투자자
- **I want to** 뉴스 API + 웹 검색으로 테마/종목별 최신 트렌드 분석 보고서를 자동 생성받고 싶다
- **So that** "왜 이 테마/종목인지" 근거를 확보할 수 있다

## 7. 비기능적 요구사항

- **성능**: 테마 추천 결과는 3초 이내 표시
- **보안**: 사용자 투자 데이터는 암호화 저장
- **확장성**: 소규모(동시 접속 ~50명)로 시작, 이후 확장 가능한 구조
- **가용성**: 장 시간(09:00~15:30) 중 99% 가동률
- **반응형**: PC와 모바일 모두 최적화된 UI

## 9. 해결 가설

### 가설
"테마별 종목을 11개 지표로 다중 정렬·비교할 수 있다면, 투자자는 대장주를 빠르게 식별하고 확신 있게 투자 결정을 내릴 수 있다."

### 실험 방법
1. MVP 출시 후 2주간 매일 사용
2. TAP으로 선택한 종목 vs 직관으로 선택한 종목의 수익률 비교
3. 테마 분석 소요 시간 측정 (기존 vs TAP 사용 후)

### 학습 목표
- 어떤 지표가 가장 자주 정렬되는가? (사용자 행동 패턴)
- 대장주 식별 정확도는 얼마나 향상되는가?

### 다음 가설
- 만약 성공하면 → FEAT-3(AI 보고서) 추가로 확신도 더 높이기
- 만약 실패하면 → 지표 종류 재검토 또는 추천 알고리즘 개선

## 11. 워크플로 (사용자 여정)

### 테마 → 종목 분석 전체 흐름

#### 1단계: 테마 분석
1. 증권회사 API에서 테마와 테마에 속한 주식 정보를 불러온다
2. 테마 추천 옵션 선택:
   - **거래량**: 가장 거래량이 활발한 테마 순위를 선정하고, 상위 5개 테마 추천
   - **급등주**: 최근 급등주가 많은 테마 5개 추천
3. 원하는 테마를 수동으로 추가할 수도 있음
4. 추천받은 테마들을 바구니에 담기

#### 2단계: 주식 분석
1. 바구니에 담긴 테마를 불러온다
2. 담긴 테마와 관련한 뉴스 API와 웹 검색을 통해 해당 테마에 대한 최신 트렌드를 분석하여 보고서를 만든다 (테마별로 각각 선택하여 제작)
3. 각 테마별 대표 대장주(거래량 상위 5개) + 관련 ETF(거래량 상위 3개)를 추천받는다
4. 각 종목별 상세 정보를 확인한다 (데이터 표로 나타나고, 각 필드별로 엑셀처럼 다중 정렬 가능)
5. 원하는 종목을 선택하여 분석 보고서를 제작
